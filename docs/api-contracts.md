# API Contracts: ICORF Odoo System

Dokumen ini mencantumkan endpoint controller kustom yang tersedia dalam modul `icofr`.

## 🖥️ Dashboard API

### 1. Get Dashboard Stats
Mengambil ringkasan data statistik untuk ditampilkan pada dashboard ICORF.

- **URL:** `/icofr/dashboard/data`
- **Method:** `POST` (JSON-RPC)
- **Auth:** `user`
- **Response Data:**
  - `total_controls`: Jumlah total master kontrol.
  - `effective_controls`: Jumlah kontrol dengan rating efektivitas 'High'.
  - `total_risks`: Jumlah total risiko teridentifikasi.
  - `high_risks`: Jumlah risiko dengan level 'High' atau 'Very High'.
  - `control_summary`: Array 5 kontrol teratas (Nama, Efektivitas, Next Test Date).
  - `scheduled_testing`: Daftar pengujian terjadwal dalam 30 hari ke depan.

## 📥 Export API

### 1. Export Control Data
Menghasilkan file CSV berisi seluruh data master kontrol untuk audit eksternal.

- **URL:** `/icofr/export/control_data`
- **Method:** `GET`
- **Auth:** `user`
- **Response:** File download (`icofr_control_data.csv`)
- **Fields:** Nama, Kode, Jenis, Frekuensi, Pemilik, Status, Efektivitas, Tanggal Pengujian.
