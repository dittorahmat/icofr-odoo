# ICORF Demo Module

## Overview
The ICORF Demo module provides sample data for demonstrating the ICORF (Internal Controls Over Financial Reporting) functionality. This module is designed to showcase all the features and scenarios outlined in the ICORF Demo Scenarios document.

## Purpose
This module is intended for:
- Training new users on the ICORF system
- Demonstrating system capabilities to stakeholders
- Providing a realistic dataset for evaluation and testing
- Facilitating hands-on learning sessions

## Data Provided
The module includes sample data based on the seven scenarios from the ICORF Demo Scenarios document:

1. **Initial Implementation and Control Mapping**: Business processes for cash management, payroll, etc.
2. **Specific Process Controls**: Controls for cash receipt and disbursement processes
3. **Certification Process**: Sample certifications and POJK compliance reports
4. **Audit Findings Response**: Sample findings and corrective action plans
5. **Calendar and Scheduling**: Testing schedules and calendar events
6. **Reporting and Analysis**: Sample reports with metrics and analytics
7. **Remediation Process**: Negative findings and remediation workflows

## Usage
1. Install the `icofr_demo` module alongside the `icofr` module
2. The demo data will be automatically loaded when the module is installed
3. Navigate through the ICORF menus to explore the sample data
4. Use the sample data to practice different workflows and understand system capabilities

## Dependencies
- This module requires the `icofr` module to be installed
- Standard Odoo `base` module
