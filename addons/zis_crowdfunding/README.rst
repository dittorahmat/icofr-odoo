==================
ZIS Crowdfunding
==================

Module ini menyediakan platform crowdfunding khusus untuk penggalangan dana ZIS (Zakat, Infak, Shodaqoh) menggunakan Odoo.

Fitur Utama
===========

* Pembuatan kampanye ZIS dengan target dana
* Sistem donasi online untuk zakat, infaq, dan shodaqoh
* Klasifikasi jenis ZIS dan kategori mustahiq
* Penyaluran dana ZIS ke penerima yang tepat
* Dashboard untuk pengelola ZIS
* Laporan keuangan ZIS
* Transparansi penyaluran dana
* Integrasi dengan website untuk tampilan depan

Instalasi
=========

1. Tempatkan modul ini di folder addons Odoo
2. Aktifkan modul melalui menu Apps (pastikan Developer Mode aktif)
3. Cari dan instal modul "ZIS Crowdfunding"

Konfigurasi
===========

* Setelah instalasi, buka menu ZIS Management untuk mengelola kampanye, donasi, dan penyaluran
* Tambahkan kategori mustahiq sesuai kebutuhan
* Buat kampanye ZIS baru dengan menentukan jenis dan target dana

Penggunaan
==========

1. Membuat Kampanye ZIS:
   * Buka menu ZIS Management > Campaigns > ZIS Campaigns
   * Klik "Create" dan isi detail kampanye
   * Pilih jenis ZIS (zakat, infaq, shodaqoh) dan kategori mustahiq

2. Melakukan Donasi:
   * Donatur dapat mengakses halaman kampanye melalui website
   * Pilih kampanye dan klik "Donasi Sekarang"
   * Isi form donasi dan submit

3. Menyalurkan Dana:
   * Buka menu ZIS Management > Distributions
   * Buat entri penyaluran baru dengan menentukan penerima dan kategori

Catatan
=====

Modul ini dirancang untuk memenuhi kebutuhan pengelolaan ZIS sesuai prinsip syariah.
Pastikan untuk melakukan validasi terhadap mustahiq sebelum menyalurkan dana.
